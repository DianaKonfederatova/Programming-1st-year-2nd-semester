# Программирование (1-ый курс, 2-ой семестр)

Решения лабораторных работ по программированию. 

## 1-ый курс
Курс **«Программирование»** охватывает:
- Основы языков С и С++
- Работа со статическими и динамическими массивами
- Работа с глобальными и локальными переменными
- Перегрузка функций. Указатели на функции. Передача функции в качестве параметра
- Работа со строками
- Структуры данных
- Классы памяти
- Односвязные и двусвязные списки
- Текстовые и бинарные файлы
- Многофайловый проект

## Требования для компиляции и запуска программ

- Компилятор C 
- Компилятор C++ 

## Сборка и запуск

```bash
# Перед компиляцией и запуском перейти на нужную папку!:
cd labX

# Если используется бибилиотека math.h, то в конце команды компиляции добавлять -lm, иначе код не скомпилируется! 

# Для C:
gcc labX.c -o labX 
./labX

# Для C++:
g++ labX.cpp -o labX
./labX

# Для lab12 компиляция и запуск:
g++ main.cpp sort.cpp -o lab12
./lab12
